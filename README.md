Project Folder Setup

Before running the project, you must create the following folders inside the root directory:

project-root/
│
├── data/
│   ├── raw/
│   └── processed/
│
├── models/
│
├── trial_backtest/
│
├── src/
│
├── notebooks/

Folder Details
data/

This folder must contain two subfolders:

raw → Store original datasets

processed → Store cleaned and processed datasets

models/

Use this folder to save all trained models.

trial_backtest/

Use this folder to store the outcomes and results of trial runs and backtesting.

src/

Place all source code files in this folder.

notebooks/

Place all Jupyter notebooks in this folder.
